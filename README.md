# 🍏 iPad

아이패드 제품 페이지를 만든 예제로 아래 명시된 기술을 활용했습니다.

- 오픈 그래프(The Open Graph protocol)
- 트위터 카드(Twitter Cards)
- 구글 폰트
- 이미지 스프라이트(Sprite Icon Animation)
- IntersectionObserver
- Vercel
